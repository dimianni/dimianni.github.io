<h1 align="center">My Portfolio</h1>
<p>It contains my most recent knowledge and experience.</p>


<h2 align="center">In order to run it locally, clone the repository and run the following command in the terminal: </h2>

```bash
  npm i 
```

<h2 align="center">Commands:</h2>

This will start a development server:
```bash
npm run start
```

This will create a production build (dist folder):
```bash
npm run build
```
