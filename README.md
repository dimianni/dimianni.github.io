## My portfolio website
It is build upon [my webpack configuration](https://github.com/dimianni/my-webpack-configuration) and contains my most recent knowledge and experience. I didn't use jQuery for this project, and tried to rely only on modern JavaScript features such as promises, let/const, arrow functions, import/exports, fetch, classes and so forth.

## To clone the repo, run the following command:

```bash
git clone https://github.com/dimianni/dimianni.github.io.git
```

## Usage

There are only two commands.

This will start a development server:
```bash
npm run start
```

This will create a production build:
```bash
npm run build
```

## Dependencies
There are only four dependencies in this project:

- [fontfaceobserver](https://www.npmjs.com/package/fontfaceobserver), which is used to preload the fonts.
- [imagesloaded](https://www.npmjs.com/package/imagesloaded), which is used to preload the images.
- [normalize.css](https://www.npmjs.com/package/normalize.css) for CSS normalization between browsers.
- [gsap](https://www.npmjs.com/package/gsap), which is the heart of almost all animations on the website.
